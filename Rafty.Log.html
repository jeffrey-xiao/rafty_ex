    rafty v0.1.0 Rafty.Log behaviour View Source

    A behavior that defines the log interface that the Raft protocol uses to persist data.

    Link to this section Summary

    Types

    state()

    The state of the log.

    Functions

    merge_logs(old, new)

    Merges two lists of logs together. All entries after the first conflict are removed from old and replaced with the entries from new. If there are no conflicts, and new is missing entries, then this function is a no-op.
