    rafty v0.1.0 API Reference

    Modules

    Rafty

    An implementation of the Raft consensus algorithm written in pure Elixir. For learning purposes only and not production use.

    Rafty.FSM

    A behavior that defines the finite state machine interface that the Raft protocol modifies and reads from.

    Rafty.FSM.Server

    A server that is the interface to the underlying Raft state machine.

    Rafty.Log

    A behavior that defines the log interface that the Raft protocol uses to persist data.

    Rafty.Log.Entry

    An entry of a Raft log.

    Rafty.Log.InMemoryStore

    Implementation of Rafty.Log that stores the entries and metadata in-memory. This store should only be used for testing and might yield incorrect results if servers crash and lose data that must be persisted for correctness.

    Rafty.Log.Metadata

    Persisted metadata of a Raft log.

    Rafty.Log.RocksDBStore

    Implementation of Rafty.Log that stores the entries and metadata in RocksDB.

    Rafty.Log.Server

    A server that is the interface to the underlying Raft log.

    Rafty.RPC

    A module that handles sending and broadcasting remote procedure calls (RPCs) specified by the Rafty protocol.

    Rafty.RPC.AppendEntriesRequest

    A request for appending entries to the Raft log.

    Rafty.RPC.AppendEntriesResponse

    A response to Rafty.RPC.AppendEntriesRequest.

    Rafty.RPC.RequestVoteRequest

    A request for a vote in a Raft election.

    Rafty.RPC.RequestVoteResponse

    A response to Rafty.RPC.RequestVoteRequest.

    Rafty.Server

    A server that implements the Raft consensus protocol.

    Rafty.Server.ClientRequest

    Rafty.Server.ClientRequest is a request from a client to a Raft group.

    Rafty.Server.Supervisor

    Supervisor for a server. It starts a Rafty.Log.Server, Rafty.FSM.Server, and a Rafty.Server.

    Rafty.ServersSupervisor

    Dynamic supervisor that supervises Rafty.Server.Supervisor.

    Rafty.Supervisor

    Supervisor for the Rafty application. It starts Rafty.RPC.Supervisor and Rafty.ServersSupervisor.

    Rafty.Timer

    A general-purpose timer that will send a message to self() after a specified duration has elapsed.
