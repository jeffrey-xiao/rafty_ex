    rafty v0.1.0 Rafty.Timer View Source

    A general-purpose timer that will send a message to self() after a specified duration has elapsed.

    Link to this section Summary

    Types

    t()

    Functions

    new(command)

    Creates a new, unset timer that sends command to self() after a specified duration has elapsed.

    reset(timer, timeout)

    Resets the timer with a new timeout. The old timeout may still trigger a message, so you must check that the reference sent in the message is ref stored in the timer. If the references don't match, then the message must be ignored.

    stop(timer)

    Stops the timer. The old timeout may still trigger a message, so you must check that the reference sent in the message is ref stored in the timer. If the references don't match, then the message must be ignored.
